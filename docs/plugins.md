# 插件

SQL Studio 的插件入口位于顶部工具栏的“🔌 插件管理”。首个插件为钉钉自定义机器人消息。

## 钉钉机器人

1. 在钉钉群中创建自定义机器人，并选择“加签”安全设置。
2. 打开插件管理，填写完整 Webhook 和以 `SEC` 开头的加签密钥。
3. 点击“安全保存”，再使用“发送测试消息”验证配置。

Webhook 中的 `access_token` 和加签密钥均属于敏感信息。应用将二者原子保存到 Windows 凭据管理器，不写入 `store.json`，读取配置状态时也不会把原值返回页面。

发送由 Rust 宿主完成，仅接受 `https://oapi.dingtalk.com/robot/send`，拒绝非官方主机、非 HTTPS、重定向、异常端口和错误路径。每次发送即时生成时间戳与 HMAC-SHA256 签名，不自动重试，避免超时场景产生重复消息。

删除配置会同时清除 Windows 凭据管理器中的 Webhook 与加签密钥。
