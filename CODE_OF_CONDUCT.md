# 社区行为准则

## 我们的承诺

我们承诺为所有参与者提供开放、友善、无骚扰的协作环境，不因年龄、体型、残障、族裔、性别特征、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教或性取向而区别对待。

## 倡导的行为

- 使用友善、包容和尊重的语言；
- 尊重不同观点、经验和技术背景；
- 接受建设性反馈，并聚焦事实和代码；
- 对社区和用户负责，主动保护隐私与敏感数据；
- 在冲突中寻求澄清和可验证的解决方案。

## 不可接受的行为

- 人身攻击、侮辱、威胁、跟踪或骚扰；
- 发布他人的私密信息、凭据、内部系统数据或身份信息；
- 性化内容或不受欢迎的性关注；
- 蓄意破坏讨论、垃圾信息或持续偏离主题；
- 其他在专业协作环境中明显不当的行为。

## 适用范围

本准则适用于仓库、Issue、Pull Request、代码评审、发布渠道，以及成员以项目代表身份参与的其他公开或私密空间。

## 执行

请通过仓库的私密漏洞报告功能或维护者在托管平台公开的私密联系方式报告行为问题。不要在公开 Issue 中披露受害者身份或私密材料。

维护者会根据影响采取纠正、警告、临时限制或永久移除等措施，并在处理过程中尽量保护报告者隐私。维护者自身违反本准则时，同样应回避相关处理。

## 归属

本准则参考 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)，并按本项目的协作方式进行了简化。
